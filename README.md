### Para empezar

- Asegúrate de tener Node.js y npm instalados en tu sistema.
- Instala las dependencias del proyecto mediante el siguiente comando:
  ```sh
  npm install
- Para iniciar la aplicación, ejecuta alguno de los siguientes comandos:
  ```sh
  npm start
  # o
  npm run dev
  ```
